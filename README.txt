✏️ КАК ДОБАВИТЬ НОВЫЕ ВОПРОСЫ:

1. Откройте questions.js в любом текстовом редакторе

2. Найдите нужный раздел (Geography, English, Math, German, Science)

3. Скопируйте этот шаблон:

    {
      prompt: "Ваш вопрос?",
      options: ["Вариант A", "Вариант B", "Вариант C", "Вариант D"],
      answer: 2,  // 0=A, 1=B, 2=C, 3=D
      explain: {
        en: "Explanation in English",
        ru: "Объяснение на русском"
      },
      points: 1
    },

4. Вставьте в конец массива вопросов (перед закрывающей скобкой ])

5. Сохраните файл

6. Обновите страницу в браузере (F5)

💡 ПРИМЕРЫ:

▶ Простой вопрос (Math):
    {
      prompt: "2 + 2 = ?",
      options: ["3", "4", "5", "6"],
      answer: 1,
      explain: {
        en: "2 + 2 = 4",
        ru: "2 + 2 = 4"
      },
      points: 1
    },

▶ Длинный вопрос (English):
    {
      prompt: "Choose the correct form: She ___ to school every day.",
      options: [
        "go",
        "goes",
        "going",
        "is go"
      ],
      answer: 1,
      explain: {
        en: "'Goes' is correct because 'she' is third person singular.",
        ru: "'Goes' правильный вариант, т.к. 'she' — третье лицо единственного числа."
      },
      points: 1
    },

⚠️ ВАЖНЫЕ ПРАВИЛА:

1. ЗАПЯТЫЕ: Каждый вопрос должен заканчиваться запятой ","

2. КАВЫЧКИ: Весь текст в двойных кавычках "текст"
   Если в тексте есть кавычки, используйте \"слово\"

3. ИНДЕКСЫ ОТВЕТОВ:
   answer: 0 → первый вариант (A)
   answer: 1 → второй вариант (B)
   answer: 2 → третий вариант (C)
   answer: 3 → четвертый вариант (D)

4. ОБЪЯСНЕНИЯ: Обязательно заполняйте поля en и ru
   Это нужно для режима обучения!

🆘 ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ:

Q: Вопросы не загружаются, страница пустая
A: Проверьте, что оба файла в одной папке и questions.js не поврежден

Q: Показывает ошибку "QBANK is not defined"
A: questions.js не загрузился. Проверьте имя файла и путь

Q: Объяснения не появляются
A: Проверьте, что у каждого вопроса есть поле "explain"

Q: Можно ли добавить новый предмет?
A: Да! Добавьте новый раздел в QBANK:
   NewSubject: [
     { вопросы... }
   ],
   И добавьте кнопку в HTML
